# 有限会社頼鉄工所 研削加工計算ポータル

有限会社頼鉄工所の研削加工支援ポータルです。社内従業員がPC、タブレット、スマホから加工条件をすばやく確認できるように、HTML、CSS、JavaScriptのみで構成しています。

このサイトはGitHub Pagesで公開する前提の静的Webサイトです。外部公開されても問題ないように、個人名、従業員情報、取引先名、図面番号、加工単価、社内機密情報は入れません。

## 現在実装済み機能

- 円筒研削条件計算
  - 砥石周速 a（m/min）
  - 砥石直径 b（mm）
  - 切込み e（mm）
  - 主軸回転数 f（rpm）
  - 荒切込（mm）
  - 精研切込（mm）
  - 仕上切込（mm）
- 重量計算
  - 鉄系材料専用
  - 丸棒
  - 中空丸棒（パイプ）
  - 固定密度 7.85 g/cm³
  - 重量（kg）

## 今後追加予定

- 内径研削条件計算
- 平面研削条件計算
- テーパー角度計算

未実装の機能は画面上で「準備中」と表示します。

## GitHub Pages公開方法

このリポジトリでは `gh-pages` ブランチをGitHub Pages公開用ブランチとして使います。

1. GitHubで `grinding-calculation-portal` リポジトリを開きます。
2. `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. `Branch` を `gh-pages`、フォルダを `/root` にします。
6. `Save` を押します。

公開URL:

```text
https://yori-3.github.io/grinding-calculation-portal/
```

## ローカル実行方法

特別なビルドは不要です。

1. このリポジトリをPCへ取得します。
2. `index.html` をブラウザで開きます。

ローカルで確認するだけなら、サーバー起動や追加ライブラリは不要です。

## 新しい計算機追加方法

1. `script.js` の `calculators` 配列にメニュー定義を追加します。
2. 未実装の場合は `status: "comingSoon"` にします。
3. 実装する場合は `status: "ready"` にし、`render` に画面描画関数を指定します。
4. 入力欄、入力チェック、計算式、結果表示を機能ごとの関数として追加します。
5. すべての数値に単位を表示します。
6. エラー表示は分かりやすい日本語にします。
7. READMEの「現在実装済み機能」「今後追加予定」「更新履歴」を更新します。

既存画面を変更する場合は、変更理由、メリット、デメリットを提示し、了承を得てから変更します。

## 円筒研削条件計算の計算式

```text
主軸回転数 f（rpm） = (60 * a / Math.PI / b) * 1000
荒切込 = f * e
精研切込 = 荒切込 / 3
仕上切込 = 荒切込 / 9
```

表示桁数:

- 主軸回転数 f: 小数1桁
- 荒切込: 小数3桁
- 精研切込: 小数3桁
- 仕上切込: 小数3桁

## 重量計算の計算式

鉄系材料専用として、密度は `7.85 g/cm³` 固定です。

丸棒:

```text
体積（mm³） = π × (D / 2)^2 × L
重量（kg） = 体積 × 7.85 ÷ 1,000,000
```

中空丸棒（パイプ）:

```text
体積（mm³） = π × ((D / 2)^2 − (d / 2)^2) × L
重量（kg） = 体積 × 7.85 ÷ 1,000,000
```

表示桁数:

- 重量: 小数3桁

注意:

本計算は鉄系材料（密度7.85g/cm³）の概算重量です。実際の重量は材質、寸法公差、穴加工などにより変動します。
