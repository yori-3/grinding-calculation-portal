# 研削加工計算ポータル

有限会社頼鉄工所向けの、加工計算を行う静的Webサイトです。

HTML、CSS、JavaScriptのみで構成しているため、GitHub Pagesや社内共有フォルダで公開できます。

## 現在の実装状況

- 円筒研削条件計算: 実装済み
- 内径研削条件計算: 準備中
- 平面研削条件計算: 準備中
- テーパー角度計算: 準備中

## ファイル構成

```text
index.html
style.css
script.js
README.md
```

## GitHub Pagesで公開する手順

1. GitHubにリポジトリを作成します。
2. `index.html`、`style.css`、`script.js`、`README.md` をリポジトリへ配置します。
3. GitHubのリポジトリ画面で `Settings` を開きます。
4. `Pages` を開き、公開元を `main` ブランチの `/root` に設定します。
5. 表示されたURLへアクセスし、画面が表示されることを確認します。

GitHub Pagesで公開した場合、URLを知っている外部の人から閲覧できる可能性があります。社内限定にしたい場合は、GitHub Pagesではなく社内共有フォルダまたは社内LANサーバーで運用してください。

## ファイルの修正方法

- 画面構成を変える場合は `index.html` を修正します。
- 色、余白、入力欄、ボタンなどの見た目を変える場合は `style.css` を修正します。
- 計算メニュー、入力チェック、計算式を変える場合は `script.js` を修正します。
- 運用説明を変える場合は `README.md` を修正します。

## 円筒研削条件計算の計算式

入力項目:

- 砥石周速 a
- 砥石直径 b（mm）
- 切込み e

計算項目:

```text
主軸回転数 f（rpm） = (60 * a / Math.PI / b) * 1000
荒切込 = f * e
精研切込 = g / 3
仕上切込 = g / 9
```

表示桁数:

- 主軸回転数 f: 小数1桁
- 荒切込: 小数3桁
- 精研切込: 小数3桁
- 仕上切込: 小数3桁

## 新しい計算機を追加する手順

1. `script.js` の `calculators` 配列に新しいメニュー定義を追加します。
2. 実装する場合は `status` を `ready` にし、`render` に表示用関数を指定します。
3. 入力欄を表示する `render...Calculator` 関数を追加します。
4. 計算処理用の関数を追加します。
5. 共通デザインで足りない見た目だけ `style.css` に追加します。
6. READMEの実装状況と計算式を更新します。

準備中のまま表示したい場合は、`status` を `comingSoon` にします。

## フォルダ内に配置して公開する場合

GitHub Pagesの公開元をリポジトリ直下にする場合は、`index.html`、`style.css`、`script.js`、`README.md` をリポジトリ直下に置いてください。

もし `頼鉄工所_加工計算サイト` フォルダ内に置いたまま公開する場合は、GitHub PagesのURL末尾に `/頼鉄工所_加工計算サイト/` を付けてアクセスします。ただし、運用を分かりやすくするため、正式公開ではリポジトリ直下に4ファイルを置く構成を推奨します。

## 社内向け注意事項

- このサイトは加工計算ポータルです。
- 現時点では円筒研削条件計算のみ実装しています。
- データ保存機能はありません。
- 入力した値はブラウザ内で計算するだけで、サーバーへ送信しません。
- GitHub Pagesで公開する場合は外部から閲覧可能になる点に注意してください。
- サイト内には、個人名、従業員情報、取引先名、図面番号、加工単価、社内機密情報を入れないでください。
